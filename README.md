## What I Work On

### 1. 메일 시스템 개발 및 운영
메일 솔루션 환경에서 사용자, 인증, 동기화, 스케줄링과 관련된 기능을 개발하고 운영합니다.

예를 들어,
- 사용자 동기화 및 삭제 처리
- 메일 송수신 관련 설정 관리
- SMTP 인증 및 릴레이 허용 로직 정리
- 만료 예정 안내 메일 발송 및 실제 데이터 정리 스케줄러 구현
- 내부/외부망 분리 환경에서의 메일 동기화 구조 고민
- UID 기반 메일 매핑 및 삭제 연동 처리

단순히 기능을 구현하는 데서 끝나지 않고,  
**통신 실패, 예외 데이터, 클라이언트별 비정상 포맷 같은 운영 이슈를 어떻게 견고하게 처리할지**에 집중하고 있습니다.
