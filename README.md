# Portfolio

최서원님의 **개인 포트폴리오 웹사이트**입니다. 백엔드 개발·정보보안 역할을 소개하고, 프로젝트와 연락처를 한 페이지에서 볼 수 있도록 구성했습니다.

## 기술 스택

- **HTML5** — 시맨틱 마크업, 접근성 속성(`aria-*`) 일부 적용
- **CSS3** — 커스텀 속성(디자인 토큰), Flexbox·Grid, 반응형 미디어 쿼리
- **Vanilla JavaScript** — 빌드 도구 없이 동작

외부 리소스: Google Fonts(Noto Sans KR, DM Sans, Instrument Serif), Unsplash 이미지.

## 주요 기능

| 구역 | 설명 |
|------|------|
| **헤더** | 고정 헤더, 로고·내비게이션, 스크롤 시에도 상단 유지 |
| **히어로** | 전체 화면 배경 이미지, 어두운 오버레이, 인사 문구·역할(백엔드 개발자), CTA 버튼 |
| **About** | 경력·관심 분야 소개(백엔드·정보보안) |
| **Projects** | 프로젝트 카드 3개(썸네일 이미지, 태그, 제목, 설명, 링크) |
| **Contact** | 이름·이메일·제목·메시지 폼, 클라이언트 측 유효성 검사, 제출 시 안내 메시지(데모) |
| **푸터** | 연도, GitHub·LinkedIn 링크 |

### 인터랙션·UX

- **반응형**: PC·태블릿·모바일 레이아웃 대응
- **모바일 메뉴**: 햄버거 버튼으로 내비 열기·닫기, `Escape`로 닫기
- **부드러운 스크롤**: 앵커 링크 클릭 시 헤더 높이를 고려해 섹션으로 이동
- **접근성**: 포커스 스타일, 폼 오류 시 `role="alert"` 영역에 메시지 표시
- **애니메이션**: `prefers-reduced-motion` 환경에서는 과한 움직임 축소

## 파일 구조

```
portfolio/
├── index.html    # 페이지 구조·콘텐츠
├── styles.css    # 전역 스타일·테마·반응형
├── script.js     # 내비, 스크롤, 폼 검증·제출 처리
└── README.md
```

## 로컬에서 보기

정적 파일이므로 아무 HTTP 서버로 열어도 됩니다.

```bash
# Python 3
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 으로 접속합니다. `index.html`을 파일로 직접 열어도 대부분 동작하지만, 일부 브라우저에서 로컬 파일 제한이 있을 수 있어 서버 사용을 권장합니다.

## 커스터마이징 포인트

- **문구·이름**: `index.html`의 히어로, About, 프로젝트 카드, 메타 태그
- **히어로 배경**: `.hero-bg-image`의 `src`
- **프로젝트 썸네일**: 각 카드 내부 `<img>`의 `src`·`alt`
- **색·간격**: `styles.css` 상단 `:root` 변수
- **폼 실제 전송**: `script.js`의 `submit` 핸들러에 백엔드 API, Formspree 등 연동
